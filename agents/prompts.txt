SECTION 6: FILTER VALUES EXTRACTION (FROM REWRITTEN QUESTION)
═══════════════════════════════════════════════════════════════════

**IMPORTANT:** Extract filter values from the REWRITTEN question (after applying context inheritance), not from the current user input.

**THREE CORE RULES:**

1. **EXCLUDE if value has attribute label**: "carrier MDOVA" → exclude "MDOVA", "invoice #76273623" → exclude "76273623", "member ID 12345" → exclude "12345"
   - Attribute labels: carrier, invoice #, invoice number, claim #, claim number, member ID, member number, patient ID, provider ID, client, account, transaction ID, reference number, contract number, policy number

2. **EXCLUDE all pure numbers**: Any digits-only value (e.g., "76273623", "982967283632", "12345", "2024", "100")

3. **INCLUDE only standalone strings/alphanumeric**: Extract values with letters that appear without attribute labels
   - ✅ Strings: "covid vaccine", "diabetes", "California"
   - ✅ Alphanumeric: "ABC123", "COVID-19" (if standalone)
   - ❌ Pure numbers: "12345", "76273623"

**INCLUDE (Standalone values only):**
- Product/drug names, therapy classes, medical conditions, geographic locations, medical specialties, brands, pharmaceutical companies
- **ONLY if standalone (no attribute labels like "carrier", "invoice #", "member ID", "client")**
- Multi-word phrases: Keep together → "covid vaccine" as ["covid vaccine"]
- Multiple terms: Separate → "diabetes, asthma" as ["diabetes", "asthma"]

**EXCLUDE (DO NOT extract):**
- **Attribute-labeled values**: carrier/invoice #/claim #/member ID/client/account/provider ID + value
- **Pure numbers**: digits-only values (any length)
- **Common terms**: "HDP", "Home Delivery", "SP", "Specialty", "Mail", "PBM", "Claim Fee", "Claim Cost", "Activity Fee"
- **Time/Date**: months, quarters, years, dates, temporal references
- **Generic**: "revenue", "cost", "expense", "data", "top", "bottom", "show", "total", "overall"
- **Metrics**: revenue, billed amount, claims count, expense, volume, cost

**EDGE CASES:**

**Multiple values with same attribute:**
"carrier MDOVA and BCBS" → exclude both "MDOVA" and "BCBS" (both have attribute "carrier")

**Mixed standalone and attributed:**
"diabetes for member ID 12345" → extract "diabetes" ✅, exclude "12345" ❌

**Alphanumeric identifiers:**
"drug ABC123" → check if "drug" is attribute label → if yes, exclude "ABC123"

**EXAMPLES:**

Example 1:
Rewritten: "What is the revenue for covid vaccines for carrier MDOVA for Q3"
→ "covid vaccines" = standalone string ✅
→ "MDOVA" = has attribute "carrier" ❌
→ filter_values: ["covid vaccines"]

Example 2:
Rewritten: "What is invoice #76273623 for diabetes drugs in California for carrier BCBS"
→ "76273623" = has attribute "invoice #" + pure number ❌
→ "diabetes" = standalone string ✅
→ "California" = standalone string ✅
→ "BCBS" = has attribute "carrier" ❌
→ filter_values: ["diabetes", "California"]

Example 3:
Rewritten: "What is claim number 982967283632 for cardiology for HDP for July 2025"
→ "982967283632" = has attribute "claim number" + pure number ❌
→ "cardiology" = standalone string ✅
→ filter_values: ["cardiology"]
(Excludes: HDP = common term, July/2025 = time)
